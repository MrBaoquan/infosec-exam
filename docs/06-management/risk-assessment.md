# 风险评估

## 风险评估基本要素

```
风险 = 资产 × 威胁 × 脆弱性
```

| 要素 | 含义 |
|------|------|
| **资产** | 有价值的信息/系统（机密性/完整性/可用性价值） |
| **威胁** | 可能导致破坏的源头（人/自然/环境） |
| **脆弱性** | 可被威胁利用的弱点（漏洞） |
| **风险** | 威胁利用脆弱性造成资产损失的可能 |
| **安全措施** | 降低风险的对策 |

> 关系：威胁利用脆弱性 → 危害资产 → 形成风险；安全措施降低脆弱性/威胁。

## 风险评估方法

| 方法 | 特点 | 代表 |
|------|------|------|
| **定性** | 矩阵分级（高/中/低） | 风险矩阵 |
| **定量** | 数值计算 | ALE 分析 |
| 综合 | 定性+定量 | — |

### 定量分析公式

| 指标 | 含义 |
|------|------|
| **AV** 资产价值 | 资产价值 |
| **EF** 暴露因子 | 单次损失比例 |
| **SLE** 单次预期损失 | SLE = AV × EF |
| **ARO** 年发生率 | 年度发生次数 |
| **ALE** 年度预期损失 | **ALE = SLE × ARO** |

> 控制措施成本 < ALE 降幅 时，措施才划算。

## 风险处置策略

| 策略 | 说明 |
|------|------|
| **降低** | 采取控制措施减小风险 |
| **转移** | 保险、外包（不消除风险，转嫁损失） |
| **规避** | 停止风险活动 |
| **接受** | 风险可接受，不处理 |

## 评估流程

1. 资产识别与赋值
2. 威胁识别
3. 脆弱性识别（漏洞扫描/渗透）
4. 风险分析计算
5. 风险处置建议
6. 持续监控

## 考点速记

- **风险 = 资产 × 威胁 × 脆弱性**
- **ALE = SLE × ARO = AV × EF × ARO**
- 处置四策略：**降低/转移/规避/接受**
- 转移=保险（不消除风险）；规避=停止活动
- 控制措施成本应 < ALE 降幅
