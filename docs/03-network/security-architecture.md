# 前沿安全架构与技术

> 本篇覆盖大纲中"网络安全前沿技术发展动向"及现代安全架构内容。标注 ⭐ 的是高频考点，标注 📌 的是了解即可。

## 零信任架构（Zero Trust）⭐

> 🎯 **核心理念**："从不信任，始终验证"——不再区分内外网，所有访问都需认证授权。

### 传统边界安全 vs 零信任
| 对比 | 传统边界安全 | 零信任 |
|------|-------------|--------|
| 信任模型 | 内网可信，外网不可信 | **从不信任** |
| 边界 | 防火墙为边界 | 无边界 |
| 认证 | 一次登录 | **持续验证** |
| 授权 | 粗粒度 | **最小权限+动态** |

### 零信任核心原则
1. **持续验证**：不只登录时验证，访问过程中持续验证身份和上下文
2. **最小权限**：只授予完成任务所需最小权限
3. **默认拒绝**：未明确允许的就拒绝
4. **微隔离**：将网络划分为细粒度区域，限制横向移动

### SDP 软件定义边界（Software-Defined Perimeter）📌
- 零信任的一种实现方式
- 隐藏目标资源，只有认证后才能访问
- BeyondCorp 是 Google 的零信任实现

### 微隔离（Microsegmentation）📌
- 将内网划分为极细粒度的安全区域
- 限制**东西向流量**（服务器间横向移动）
- 南北向流量=进出数据中心的流量；东西向流量=数据中心内部流量

## ATT&CK 框架 ⭐

> MITRE ATT&CK 是攻击者行为知识库，描述攻击者的战术(Tactics)和技术(Techniques)。

### Kill Chain（攻击杀伤链）⭐
洛克希德·马丁提出的7阶段攻击模型：
```
侦察 → 武器化 → 投递 → 利用 → 安装 → 命令控制(C2) → 行动
```

| 阶段 | 说明 | 防御 |
|------|------|------|
| 侦察 | 收集目标信息 | 信息最小化 |
| 武器化 | 制作攻击载荷 | — |
| 投递 | 发送载荷(钓鱼/漏洞) | 邮件过滤/WAF |
| 利用 | 触发漏洞 | 补丁/加固 |
| 安装 | 植入后门 | EDR/白名单 |
| 命令控制(C2/C&C) | 远程控制 | 流量检测/DNS监控 |
| 行动 | 窃取/破坏 | DLP/监控 |

### ATT&CK vs Kill Chain
- Kill Chain：7阶段线性模型
- ATT&CK：更细粒度的矩阵，覆盖更多战术和技术

## 威胁情报（Threat Intelligence）⭐

> 威胁情报(CTI)是关于攻击者的知识，用于**提前预警和防护**。

| 类型 | 说明 |
|------|------|
| **战略情报** | 高级趋势（谁在攻击什么行业） |
| **战术情报** | 攻击者TTPs（战术/技术/流程） |
| **运营情报** | 具体攻击事件预警 |
| **技术情报** | IOC指标（IP/域名/哈希） |

### IOC（妥协指标）⭐
- 恶意IP、域名、文件哈希、注册表键等
- 用于检测是否被入侵
- 格式：STIX/TAXII（情报共享标准）

### 情报共享标准 📌
- **STIX**：情报描述语言
- **TAXII**：情报传输协议
- **MISP**：开源情报共享平台

## 安全运营体系

### SIEM（安全信息与事件管理）⭐
- 集中收集日志/事件 → 关联分析 → 告警
- 代表：Splunk、ELK+Elastic Security
- 核心：**关联分析**发现复合攻击

### SOC（安全运营中心）⭐
- 7×24小时安全监控响应团队
- 职能：监控→检测→分析→响应→恢复

### EDR/XDR 📌
| 类型 | 说明 |
|------|------|
| **EDR** | 端点检测与响应（终端级） |
| **XDR** | 扩展检测与响应（跨端点+网络+云） |
| **NTA/NDR** | 网络流量分析/检测 |

### SOAR 📌
- 安全编排自动化与响应
- 用剧本(playbook)自动化处理告警

### 4A 体系 ⭐
| A | 说明 |
|---|------|
| 账号(Account) | 统一账号管理 |
| 认证(Authentication) | 统一身份认证 |
| 授权(Authorization) | 统一权限管理 |
| 审计(Audit) | 统一安全审计 |

## 考点速记
- 零信任："**从不信任，始终验证**"，无边界，持续验证
- Kill Chain 7阶段：侦察→武器化→投递→利用→安装→**C2**→行动
- 威胁情报IOC：恶意IP/域名/哈希；共享标准STIX/TAXII
- SIEM=日志集中+关联分析；SOC=7x24监控响应
- EDR终端级；XDR跨端点+网络+云
- 4A=账号/认证/授权/审计
- 东西向流量=内部横向；南北向=进出数据中心
- 端口扫描Nmap；Web代理Burp Suite；SQL注入sqlmap
