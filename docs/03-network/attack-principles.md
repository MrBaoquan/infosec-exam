# 网络攻击原理与常用方法

> 🎯 **形象比喻**：网络攻击像"入室盗窃"——先踩点(信息收集)→找漏洞(挖洞)→破门(获取权限)→隐藏(隐蔽)→偷东西(实施)→擦痕迹(清除)。

## 网络攻击模型

通用攻击模型：**攻击者 → 攻击工具 → 漏洞 → 后门 → 效果**

```
攻击者动机 → 信息收集 → 漏洞挖掘 → 权限获取 → 攻击实施 → 后门 → 痕迹清除
```

## 网络攻击一般过程（8步，高频）

| 步骤 | 说明 | 常用手段 |
|:----:|------|----------|
| ① **隐藏攻击源** | 隐藏真实IP | 代理跳板、肉鸡、VPN |
| ② **收集目标信息** | 踩点 | DNS查询、端口扫描、社工 |
| ③ **挖掘漏洞** | 找弱点 | 漏洞扫描、代码审计 |
| ④ **获取访问权限** | 破门 | 口令破解、漏洞利用 |
| ⑤ **隐蔽攻击行为** | 藏匿 | Rootkit、清除日志 |
| ⑥ **实施攻击** | 达成目的 | 窃取数据、篡改、破坏 |
| ⑦ **开辟后门** | 留通道 | 后门程序、WebShell |
| ⑧ **清除攻击痕迹** | 擦除 | 删日志、清历史 |

## 常见攻击技术方法

### 信息收集类
| 技术 | 说明 | 工具 |
|------|------|------|
| **端口扫描** | 探测开放端口 | Nmap |
| **漏洞扫描** | 自动找漏洞 | Nessus |
| **网络嗅探** | 抓包窃听 | Wireshark、Tcpdump |

### 权限获取类
| 技术 | 说明 |
|------|------|
| **口令破解** | 暴力/字典/彩虹表 |
| **缓冲区溢出** | 覆盖返回地址执行恶意代码 |
| **SQL注入** | 输入拼入SQL篡改查询 |

### 欺骗类
| 技术 | 说明 |
|------|------|
| **网站假冒** | 伪造合法网站 |
| **网络钓鱼** | 邮件/短信诱导 |
| **ARP欺骗** | 伪造ARP应答劫持 |
| **DNS欺骗** | 篡改DNS解析 |

### 拒绝服务类
| 技术 | 说明 |
|------|------|
| **DoS/DDoS** | 耗尽资源/带宽 |
| **SYN Flood** | 半连接耗尽 |
| **Smurf** | ICMP广播放大 |

### 其他
| 技术 | 说明 |
|------|------|
| **会话劫持** | 窃取已登录会话 |
| **社交工程** | 利用人性弱点 |
| **恶意代码** | 病毒/木马/蠕虫 |

## 黑客常用软件分类

| 类别 | 代表工具 |
|------|----------|
| 扫描类 | Nmap、Nessus、AWVS |
| 远程监控 | 冰河、灰鸽子 |
| 密码破解 | John the Ripper、Hydra |
| 网络嗅探 | Wireshark、Tcpdump |

## 考点速记
- 攻击8步：**隐藏源→收集信息→挖漏洞→获取权限→隐蔽→实施→后门→清痕迹**
- 缓冲区溢出：覆盖**返回地址**执行恶意代码
- 会话劫持：窃取**已登录会话**
- 端口扫描工具：**Nmap**；漏洞扫描：**Nessus**
- 攻击第一步通常是**隐藏攻击源**（代理/肉鸡）
- 社交工程针对**人性**而非技术
