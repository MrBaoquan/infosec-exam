## Web 安全三大攻击（下午大题高频）

### SQL 注入
- 原理：用户输入拼入 SQL，篡改查询逻辑
- 防护：**参数化查询/预编译**、ORM、输入过滤、最小权限、WAF

<SqlInjection />

### XSS 跨站脚本
- 类型：**存储型**（存数据库）/ **反射型**（URL 参数）/ **DOM 型**
- 危害：盗 Cookie、会话劫持、挂马
- 防护：**输出编码**、CSP、HttpOnly Cookie、输入过滤

### CSRF 跨站请求伪造
- 原理：借用户已登录身份发起请求
- 防护：**Token**、Referer 校验、SameSite Cookie

| 攻击 | 关键防护 |
|------|----------|
| SQL 注入 | 参数化查询 |
| XSS | 输出编码 + CSP |
| CSRF | Anti-CSRF Token |

## 其他 Web 攻击

- **文件上传**：校验类型/MIME/重命名/隔离目录
- **文件包含**：禁止动态包含，白名单
- **命令注入**：避免拼 shell，用参数化 API
- **SSRF**：限制内网访问白名单
- **越权**：垂直(权限提升)/水平(同级别互访)，每次操作校验权限

## OWASP Top 10

注入、失效认证、敏感数据泄露、XXE、失效访问控制、安全配置错误、XSS、不安全反序列化、使用含已知漏洞组件、日志监控不足。

## 数据库安全

- 视图隔离、最小权限、加密(透明加密 TDE)、审计
- 备份：全量+增量+差异；事务日志

## 邮件安全

- **PGP** 端到端加密(信任网)
- **S/MIME** 基于 PKI/证书
- **SPF/DKIM/DMARC** 防伪造发件人

## 移动安全

- Android 沙箱/签名/权限；iOS 严格沙箱
- 加固：代码混淆、反调试、完整性校验

## 云安全

- **责任共担模型**：云厂商保基础设施，用户保数据/应用
- CSA 顶威胁：数据泄露、身份/凭证管理、不安全接口

## 考点速记

- SQL 注入防 = **参数化查询**；XSS 防 = **输出编码+CSP**；CSRF 防 = **Token**
- XSS 三型：存储/反射/DOM
- 越权：垂直=提权，水平=同级别互访
- PGP 信任网，S/MIME 基于 PKI
- 云安全 = 责任共担
