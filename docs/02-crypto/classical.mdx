# 古典密码

## 两大类型

| 类型 | 原理 | 代表 |
|------|------|------|
| **替换密码** | 字母用其他字母替换 | 凯撒、维吉尼亚、Playfair |
| **置换密码** | 字母位置重排，字母本身不变 | 栅栏密码、列置换 |

## 凯撒密码（最简单替换密码）

- 加密：$C = (M + K) \bmod 26$
- 解密：$M = (C - K) \bmod 26$
- 密钥空间仅 **25**，暴力秒破

## 维吉尼亚密码 Vigenère

- 多表替换：使用密钥词循环，每个字母用不同移位
- 曾被认为"不可破"，实际可用 **Kasiski 测试 / 重合指数法** 破解
- 安全性高于凯撒，但仍是古典密码，无法抵御现代密码分析

## 古典密码的破解

1. **暴力破解**（密钥空间小）
2. **频率分析**（英文字母 e/t/a 出现频率高）— 对单表替换致命
3. **重合指数 IC**（区分单表/多表）

## 考点速记

- 单表替换 → 频率分析破
- 多表替换 → 重合指数法破
- 凯撒密钥空间 25
- 置换密码不改字母本身，只改位置
