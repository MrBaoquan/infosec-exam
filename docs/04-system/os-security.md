# 操作系统安全机制

## OS 安全核心机制

| 机制 | 作用 |
|------|------|
| **身份认证** | 确认"你是谁"——口令、生物特征、令牌 |
| **访问控制** | 决定"你能做什么"——DAC/MAC/RBAC |
| **最小权限** | 只授予完成任务所需最小权限 |
| **安全审计** | 记录日志，可追溯 |
| **可信计算** | 从硬件根建立信任链 |

## 身份认证三因素

| 因素 | 举例 |
|------|------|
| 你知道什么 | 口令、PIN |
| 你拥有什么 | 令牌、手机、证书 |
| 你是什么 | 指纹、人脸、虹膜 |

> **双因素认证 2FA** = 两个不同因素组合，如 口令+短信验证码。

## 口令安全

- 存储用**哈希+盐(salt)**，禁止明文存储
- 盐的作用：防彩虹表、相同口令哈希不同
- 慢哈希：bcrypt/scrypt/PBKDF2 增加破解成本
- 口令策略：长度≥8、复杂度、定期更换、失败锁定

## Windows 安全机制

| 机制 | 说明 |
|------|------|
| **UAC** 用户账户控制 | 提权需确认，降低恶意提权风险 |
| **组策略 GPO** | 集中配置安全策略 |
| **BitLocker** | 全盘加密（防物理窃取后读盘） |
| **EFS** | 文件级加密（NTFS） |
| **ACL** | 自主访问控制列表 |
| **Windows Defender** | 内置防病毒 |

## Linux 安全机制

| 机制 | 说明 |
|------|------|
| **SELinux** | 强制访问控制（MAC），NSA 贡献 |
| **AppArmor** | 基于 profile 的 MAC |
| **chmod/chown** | 自主访问控制 |
| **sudo** | 受限提权（记录审计） |
| **iptables/nftables** | 内核防火墙 |
| **PAM** | 可插拔认证模块 |

## 文件系统安全

- NTFS 支持 ACL（FAT32 不支持）
- Linux 权限：`rwx` 对应 属主/属组/其他
- 特殊权限位：SUID（以属主身份执行）、SGID、Sticky
- SUID 风险：`passwd` 命令需 SUID 提权，但可被滥用

## 加固清单

1. 关闭不必要服务/端口
2. 修改默认账号/口令
3. 及时打补丁
4. 启用日志审计
5. 最小权限原则
6. 禁用 root 直接登录（用 sudo）
7. 限制远程登录 IP

## 5. 易错辨析

### 易错1：口令存储
- ❌ 明文存储（最危险）
- ❌ 仅MD5/SHA（防不住彩虹表）
- ✅ **哈希+盐(salt)**+慢哈希(bcrypt/scrypt)
- 盐的作用：相同口令哈希不同，防彩虹表

### 易错2：Windows BitLocker vs EFS
- **BitLocker**：**全盘**加密（防物理窃取硬盘）
- **EFS**：**文件级**加密（NTFS）
- 考题："防硬盘被盗后读数据"→BitLocker

### 易错3：SUID
- SUID：执行时以**文件属主**身份运行
- 典型：`passwd`命令需SUID提权（普通用户以root身份改密码文件）
- 风险：被滥用则提权

### 易错4：SELinux/AppArmor 是 MAC
- SELinux/AppArmor = **强制访问控制**(MAC)
- chmod/ACL = **自主访问控制**(DAC)
- 考题常混

## 6. 真题示例及解析

### 真题1（上午·单选）

> 用户口令安全存储应采用？
> A. 明文存储  B. Base64编码  C. 哈希+盐  D. 对称加密

**答案：C**

**解析**：口令存储用**哈希+盐**，防彩虹表。慢哈希(bcrypt)进一步增加破解成本。Base64是编码非加密，明文和对称加密都可还原。

---

### 真题2（上午·单选）

> Windows BitLocker 提供的安全功能是？
> A. 防病毒  B. 全盘加密  C. 防火墙  D. 访问控制

**答案：B**

**解析**：BitLocker是**全盘加密**，防物理窃取硬盘后读取数据。EFS是文件级加密。

---

### 真题3（上午·单选）

> Linux 中 SUID 权限的含义是？
> A. 以文件属主身份执行  B. 禁止执行  C. 以其他用户身份读  D. 仅属组可写

**答案：A**

**解析**：SUID(Set User ID)：执行时以文件属主身份运行（如passwd以root身份执行）。有提权风险。

---

### 真题4（上午·单选）

> 下列属于强制访问控制(MAC)的是？
> A. chmod  B. Windows ACL  C. SELinux  D. RBAC

**答案：C**

**解析**：SELinux是MAC（系统强制按安全标签）。chmod/ACL是DAC（所有者自主）。

## 7. 考点速记

- 认证三因素：知道/拥有/是；双因素=不同类组合
- 口令存储：**哈希+盐**，防彩虹表；慢哈希增加破解成本
- Windows：UAC 提权、BitLocker 全盘加密
- Linux：**SELinux/AppArmor 是 MAC**，chmod 是 DAC
- SUID 以属主身份执行（passwd 命令典型）
- 加固核心：最小权限 + 关闭不必要服务 + 打补丁
