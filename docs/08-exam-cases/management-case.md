# 安全管理大题

## 题型特征

通常给出**场景描述**，要求：
1. 等级保护定级
2. 风险评估计算（ALE）
3. 应急响应流程
4. 安全管理制度分析

## 例题 1：等级保护定级

> **题目**：某省级政府办公系统，存储大量公民个人信息，若被破坏可能对社会秩序造成严重损害。请确定其等保级别并说明依据。

### 标准解答

**(1) 等保级别：第三级（监督保护）**

**(2) 定级依据**：
- 受到破坏后，对**社会秩序和公共利益**造成**严重损害** → 第三级
- 省级政府系统属重要信息系统

**(3) 后续工作流程**：
① 定级：确定第三级，组织专家评审
② 备案：10 个工作日内到公安机关备案
③ 建设整改：按等保 2.0 第三级要求建设安全设施
④ 测评：选择第三方测评机构，**每年**测评一次
⑤ 运维：持续运行维护，定期复查

**(4) 第三级安全要求（三重防护）**：
- 安全通信网络：网络架构、通信加密
- 安全区域边界：边界防护、访问控制、入侵防范
- 安全计算环境：身份鉴别、访问控制、安全审计
- 安全管理中心：系统管理、安全管理、审计管理

> 💡 **定级口诀**：
> - 1级：自主保护（一般系统）
> - 2级：指导保护（一般系统，损害一般）
> - 3级：**监督保护**（重要系统，损害严重）← 高频
> - 4级：强制保护（关键系统，损害特别严重）
> - 5级：专控保护（特殊系统）

## 例题 2：风险评估 ALE 计算（高频）

> **题目**：某数据库服务器资产价值 100 万元，遭受黑客攻击的暴露因子 40%，年发生率 0.3 次/年。
> (1) 计算 SLE、ALE
> (2) 若投入 8 万元防护措施可将年发生率降至 0.1，措施是否划算？

### 标准解答

**(1) 计算 SLE 和 ALE**

$$SLE = AV \times EF = 100万 \times 40\% = 40万$$

$$ALE = SLE \times ARO = 40万 \times 0.3 = 12万/年$$

**(2) 措施划算性分析**

采取措施后：
$$ALE_{新} = SLE \times ARO_{新} = 40万 \times 0.1 = 4万/年$$

风险降低值（ALE 降幅）：
$$\Delta ALE = 12万 - 4万 = 8万/年$$

措施成本 = 8 万/年

**结论**：$\Delta ALE = 8万 \geq 措施成本 8万$，勉强划算（临界）。若考虑长期收益，建议采纳。

> 💡 **评分要点**：公式 1 分 + 计算 2 分 + 结论 1 分。**必须列公式再代入数值**。

## 例题 3：应急响应

> **题目**：某公司发现内网多台主机感染勒索病毒，文件被加密。请写出应急响应流程。

### 标准解答

按应急响应六阶段：

| 阶段 | 具体措施 |
|------|----------|
| **① 准备** | 组建应急团队、准备工具（杀软、备份） |
| **② 检测** | 确认事件范围（哪些主机感染）、分析勒索病毒类型 |
| **③ 抑制** | ① 立即断网隔离感染主机 ② 禁用共享 ③ 阻断病毒传播途径 |
| **④ 根除** | ① 清除病毒 ② 修复被利用的漏洞 ③ 重置密码 |
| **⑤ 恢复** | ① 从备份恢复数据 ② 验证系统安全后恢复业务 |
| **⑥ 总结** | ① 复盘攻击路径 ② 完善防护 ③ 更新应急预案 |

> 💡 **答题要点**：
> - 抑制阶段必须**断网隔离**（控制蔓延）
> - 恢复用**备份**（不付赎金）
> - 六阶段顺序不能错：准备→检测→抑制→根除→恢复→总结

## 答题模板

### 等保定级
```
级别判断 → 依据（损害程度）→ 流程（定级/备案/整改/测评/运维）→ 测评频率
```

### 风险计算
```
SLE = AV × EF → ALE = SLE × ARO → 措施后ALE → 对比成本 → 结论
```

### 应急响应
```
准备 → 检测 → 抑制(断网) → 根除(清毒+补漏) → 恢复(备份) → 总结
```

## 考点速记
- 等保 3 级=监督保护，损害严重，**每年测评**
- 备案：**10 个工作日内**到公安机关
- ALE = AV × EF × ARO；措施成本 < ALE 降幅才划算
- 应急六阶段：**准备/检测/抑制/根除/恢复/总结**
- 抑制=断网隔离；恢复用备份不付赎金
