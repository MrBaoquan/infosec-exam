## 下午大题答题总策略

### 1. 答题原则
- **分点作答**：用①②③编号，阅卷按点给分
- **术语准确**：用规范术语（如"参数化查询"非"防注入"）
- **宁多勿少**：不确定就多写相关点，写错不扣分
- **先易后难**：先做有把握的题，难题留时间

### 2. 通用答题模板
```
① 原理分析：说明漏洞/问题的根本原因
② 具体措施：列出2-4个具体防护措施（按优先级）
③ 补充说明：技术细节或扩展（争取额外分）
```

### 3. 高频考点清单
- 密码学：选算法、RSA手算、哈希验证、密钥管理
- 网络安全：防火墙规则、IPSec配置、抓包分析
- Web安全：SQL注入/XSS/CSRF代码审计与修复
- 安全管理：等保定级、ALE计算、应急响应流程
- 综合：攻击链分析、安全方案设计
