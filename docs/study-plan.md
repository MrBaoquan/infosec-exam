## 学习方法建议

### ✅ 高效学习法
1. **先读文档**：每篇 10-15 分钟通读，重点关注表格和"考点速记"
2. **看动画**：密码学章节务必操作动画，直观理解算法
3. **做小测**：每学完一章去[题库](/quiz)做对应分类练习
4. **练大题**：第 4 周重点练下午大题，背答题模板
5. **记错题**：错题自动进错题本，考前重做

### 📊 得分策略
| 模块 | 目标得分率 | 说明 |
|------|:----------:|------|
| 密码学 | 90%+ | 动画理解机制，计算题按步骤作答 |
| 网络安全 | 80%+ | 先定位协议层，再分析攻击和措施 |
| 应用安全 | 80%+ | 理解漏洞成因，掌握代码与配置修复 |
| 管理工程 | 70%+ | 用流程、角色、产物组织答案 |
| 应用技术 | 60%+ | 依据题干分点作答，使用规范术语 |

> 两个科目必须同时达到当次合格标准。考试形式和标准以官方当次通知为准。
